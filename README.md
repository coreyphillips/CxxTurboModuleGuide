# CxxTurboModuleGuide
A React Native C++ TurboModule Template

C++ TurboModule Docs: https://reactnative.dev/docs/next/the-new-architecture/cxx-cxxturbomodules

## Getting Started

### Step 1: Install Dependencies
```bash
yarn install
```

### Step 2: Start Application
#### For Android
```bash
yarn android
```
#### For iOS
```bash
yarn ios
```
